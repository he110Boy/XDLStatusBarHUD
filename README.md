# XDLStatusBarHUD
一个简单易用的状态栏指示器框架

### 显示成功状态栏弹窗
```objc
 [XDLStatusBarHUD showSuccess];
```

### 显示失败状态栏弹窗
```objc
[XDLStatusBarHUD showError];
```

### 显示加载状态栏弹窗
```objc
[XDLStatusBarHUD showLoad];
```

### 显示文字状态栏弹窗
```objc
[XDLStatusBarHUD showWord];
```

### 隐藏状态栏弹窗
```objc
[XDLStatusBarHUD hide];
```
